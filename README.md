# HR Analytics Dashboard | Power BI

## 📌 Overview
This project is an interactive HR Analytics Dashboard built using Microsoft Power BI. It helps analyze workforce data, employee demographics, job satisfaction, overtime, and department-level insights to support HR decision-making.

## 🎯 Objectives

- Analyze employee distribution
- Monitor workforce demographics
- Identify department-wise employee count
- Analyze education background
- Study years at company
- Compare overtime distribution
- Evaluate job satisfaction

---

## 📈 Key KPIs

- Total Employees
- Average Monthly Income
- Average Age
- Average Job Satisfaction

---

## 📊 Visualizations

- Department-wise Employees
- Job Role Analysis
- Age Distribution
- Education Field Analysis
- Years at Company Trend
- Overtime Analysis
- Monthly Income by Department
- Department Summary Table
- Job Satisfaction Gauge

---

## 🛠 Tools Used

- Microsoft Power BI
- Power Query
- DAX (Planned for Advanced KPIs)

## 📂 Dataset

IBM HR Analytics Employee Attrition Dataset
